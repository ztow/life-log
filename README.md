# Life Log 生命日志

> 每日生活记录汇总系统 —— 将视频、录音、文字、图片、PDF、邮件等内容按时间线整理为结构化MD日志。

**实际日志数据存储于本地**：`~/Documents/生活日志/`（不纳入 Git 管理）

## 设计理念

每天把手机/电脑上零散的各种文件丢进一个"待整理"目录，AI自动分类处理，按时间线排好，生成一天的日志——就像给生活写了一本自动索引的日记。

支持的内容类型：

| 类型 | 处理方式 | 示例 |
|------|----------|------|
| 视频 (.mp4/.mov/.mkv) | 每秒抽帧 → vision分析每帧内容 | 看了个科普视频 |
| 录音 (.m4a/.mp3/.wav) | 转写为完整文字 | 播客访谈、会议 |
| 照片 (.jpg/.png/.heic) | vision分析图片内容 | 收到证件照片 |
| 文字文件 (.txt/.md/.json) | 直接读取 | 笔记、备忘录 |
| PDF | 书籍仅记录书名，短内容提取摘要 | 健康知识访谈 |
| 邮件 | 复制粘贴当文字处理 | 客户邮件沟通 |
| OCR图片 | 当文字提取内容 | 扫描件、截图 |

### 核心原则

- **日志日期 = 处理日期**：默认昨天，每天早上8点定时处理。你也可以手动指定。
- **条目时间 = 文件创建时间**：按时间线排序用文件元数据，不合并。
- **归档目录 = 按日志日期**：年/月/日三级目录，所有原始文件自动归档。
- **分析深度**：具体事件说清楚（谁+什么事+过程+理由+结果），日常琐事一句话带过。不抄全文。

## 目录结构

```
~/Documents/生活日志/
├── 待整理/          # 每天把文件扔这里
│   ├── IMG_1234.jpg
│   ├── 录音.m4a
│   └── notes.txt
├── 汇总/            # 生成的MD日志文件
│   ├── 2026-07-07 生命日志.md
│   └── ...
├── 归档/            # 已处理的原始文件
│   └── 2026/
│       └── 07/
│           ├── 06/       # 7月6日归档
│           └── 07/       # 7月7日归档
```

## 环境要求

- **Hermes Agent**（Nous Research）— 用于运行AI处理
- **ffmpeg** — 视频抽帧
- **whisper** / **whisper.cpp** — 可选，用于离线录音转写

## 快速开始

### 1. 目录初始化

```bash
mkdir -p ~/Documents/生活日志/{待整理,汇总,归档}
```

### 2. 安装并运行脚本

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### 3. 加载技能

在 Hermes Agent 中加载技能：

```bash
hermes skill life-log
```

### 4. 手动触发

```bash
hermes run "整理昨天的日志"
```

或指定日期：

```bash
hermes run "整理7月15日的日志"
```

### 5. 自动定时

每天早上8点自动执行（已在技能中配置）：

直接用 `hermes cron list` 查看已注册的定时任务。

## 使用示例

### 日常流程

1. 白天随手拍照片、录语音、记文字 → 统一放到 `待整理/`
2. 第二天早上8点或手动说"整理昨天的日志"
3. AI自动：分类→分析→排序→生成日志→归档原始文件
4. 打开 `汇总/2026-07-XX 生命日志.md` 查看一天

### 一天内多条不同类型记录

```
~/Documents/生活日志/汇总/2026-07-06 生命日志.md
├── 09:36 [播客访谈] 录制血脂健康节目
├── 15:57 [视频] 观看AGI科普视频
└── [汇总统计] 2个文件
```

## 输出日志格式

```markdown
---
date: 2026-07-07
files_processed: 2
types: [视频, 录音转录]
---

# 2026-07-07 生命日志

## 09:36 [播客访谈] 标题

描述内容...

## 15:57 [视频] 视频名称

描述内容...

---

## 汇总统计

- 处理文件总数：2
- 类型分布：视频 ×1，录音转录 ×1
```

## 归档目录

处理完的文件自动归档到 `归档/YYYY/MM/DD/`，按日志日期而不是文件创建日期。

```
归档/2026/07/06/
├── 智能纪要：血脂相关健康知识访谈 2026年7月7日.pdf
└── 盲王与AGI底层架构.mp4
```

## 注意事项

- 视频最长不宜超过30分钟（帧数过多）
- 录音转写依赖 whisper 或在线API
- PDF书籍只记录书名，不处理内容
- 涉及隐私的敏感材料建议手动参与
